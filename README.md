## 在线QQ音乐播放器
> 该项目分为播放, 切歌,歌曲菜单,拖拽进度条这四个功能，该项目用到了audio标签上面的播放和暂停等一系列方法,在css部分用到了less，让代码结构更加的清晰，整个项目通过gulp前端自动化构建工具构建,让代码更加模块化，结构更加清晰
